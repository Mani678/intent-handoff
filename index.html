⚡ Intent Handoff System

Most agent systems pass state between agents. This passes intent — and tracks how it degrades with every hop. One continuous flow: intent created → race resolved → handoff executed → final decision.

HOW THE FULL FLOW WORKS

1. INTENT CREATED
Root agent wraps the task in an envelope carrying the why, constraints, and confidence score.
2. RACE RESOLVED
Multiple agents race to claim. First wins a dynamic lease scaled to task complexity. Others denied.
3. HANDOFF WITH DECAY
Winning agent passes envelope through the chain. Confidence drops 12% per hop. Biggest drop logged.
4. FINAL DECISION + TRAIL
Below 0.40 → reanchor. Above 5 hops → abort. Reason trail shows exactly where intent degraded most.

STATUS LEGEND

Healthy — proceed normally
Degraded — proceed with caution
Needs reanchor — route to human
Max hops exceeded — abort
