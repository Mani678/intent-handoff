⚡ Intent Handoff — Decay Tracker

Most agent systems pass state between agents. This prototype passes intent — and tracks how that intent degrades with every hop. When confidence drops too low, the system stops and asks a human to re-anchor before proceeding.

HOW IT WORKS

INTENT ENVELOPE
Every task carries a "why" — the original reason behind it. This travels with the task across agents.
CONFIDENCE DECAY
Each hop reduces confidence by 12%. The more agents touch a task, the more the original intent gets diluted.
DECAY THRESHOLD
Below 0.40 confidence, the system routes to a human for re-anchoring instead of blindly executing.
MAX HOPS
After 5 hops the task is aborted and escalated regardless of confidence — too much has been lost.

STATUS LEGEND

Healthy — proceed normally
Degraded — proceed with caution
Needs reanchor — route to human
Max hops exceeded — abort
