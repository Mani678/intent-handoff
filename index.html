⚡ Intent Handoff System

Most agent systems pass state between agents. This passes intent — and tracks how it degrades with every hop. One continuous flow: intent created → race resolved → handoff executed → final decision.

HOW THE FULL FLOW WORKS

1. INTENT CREATED
A root agent wraps the task in an envelope carrying the why, constraints, and a confidence score.
2. RACE RESOLVED
Multiple agents race to claim the envelope. First wins a time-bounded lease. Others are denied gracefully.
3. HANDOFF WITH DECAY
The winning agent passes the envelope through the chain. Confidence drops 12% per hop.
4. FINAL DECISION
Below 0.40 confidence → human reanchor. Above 5 hops → abort. Otherwise → proceed.

STATUS LEGEND

Healthy — proceed normally
Degraded — proceed with caution
Needs reanchor — route to human
Max hops exceeded — abort
